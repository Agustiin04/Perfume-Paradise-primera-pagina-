# Perfume Paradise 🌸 - Mi Primera Página Web

![Perfume Paradise](imagenes/logo.png)

Bienvenido a **Perfume Paradise**, mi primera página web creada con HTML y CSS puro. Este proyecto es un sitio web estático dedicado al mundo de los perfumes y fragancias. Fue desarrollado como parte de mi aprendizaje en desarrollo web frontend.

## 🌐 Descripción del Proyecto

Perfume Paradise es un sitio web simple pero completo que funciona como una **tienda virtual informativa de perfumes**. El sitio incluye múltiples páginas que permiten explorar diferentes fragancias, leer noticias relacionadas con el mundo de la perfumería, conocer más sobre la marca y contactar con la empresa.

### 📋 Estructura del Sitio

El sitio está organizado en las siguientes secciones principales:

| Sección | Archivo | Descripción |
|---------|---------|-------------|
| **Inicio** | `index.html` | Página principal con presentación del sitio |
| **Fragancias** | `fragancias.html` | Catálogo general de perfumes |
| **Perfumes Individuales** | `perfume1.html` a `perfume9.html` | Páginas detalladas de cada fragancia |
| **Noticias** | `noticias.html`, `noticias2.html`, `noticia3.html`, `noticia4.html` | Blog o sección de novedades |
| **Sobre Nosotros** | `sobrenosotrs.html` | Información de la empresa/marca |
| **Contacto** | `contactanos.html` | Formulario o información de contacto |
| **Iniciar Sesión** | `iniciarsesion.html` | Página de login (fachada visual) |
| **Nuestra IA** | `nuestraIA.html` | Sección especial sobre inteligencia artificial |
| **Foro** | `foro.html` | Espacio de comunidad (fachada visual) |
| **Error 404** | `error404.html` | Página de error personalizada |

## 🎨 Estilos y Diseño

El proyecto incluye **múltiples hojas de estilo CSS** para cada sección:

- `estilos.css` - Estilos generales del sitio
- `estilosPERFU1.css` - Estilos para páginas de perfumes
- `estilosFRA.css` - Estilos para la sección de fragancias
- `estilosNOTI.css` - Estilos para noticias
- `estilosSOBRENOS.css` - Estilos para "Sobre Nosotros"
- `estilosCONTA.css` - Estilos para contacto
- `estilosINISES.css` - Estilos para inicio de sesión
- `estilosIA.css` - Estilos para la sección de IA
- `estilosFORO.css` - Estilos para el foro
- `estilos404.css` - Estilos para página de error
